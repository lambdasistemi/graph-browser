# Core Diagram

This page shows the shared graph-browser ontology rendered from the generated Mermaid source with a deterministic renderer.

The diagram covers these core concepts:

- `gb:Dataset`
- `gb:Node`
- `gb:Group`
- `gb:EdgeAssertion`
- `gb:ExternalLink`
- `gb:Tutorial`
- `gb:TutorialStop`
- `gb:View`
- `gb:EdgeReference`
- `gb:QueryCatalog`
- `gb:NamedQuery`
- `gb:QueryParameter`
- `gb:EdgeRelation`

The relations shown include dataset membership, node grouping, tutorial stops, view edge references, query catalogs, query parameters, and edge assertion structure.
